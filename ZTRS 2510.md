One-line takeaway: **ZTRS replaces imitation learning with offline RL for a score-based E2E planner, showing that trajectory scoring can be trained without expert actions.**

# Core takeaways
- Removes imitation learning from the standard score-based pipeline.
- Uses an enumerable trajectory action space to make offline RL practical.
- Shows especially strong gains on hard-scenario evaluation.

# Method summary
ZTRS formulates E2E planning as selecting from a large discrete trajectory action set and optimizes that selector with offline RL rather than behavior cloning. Exhaustive policy optimization leverages the finite action set to compute reward-improving updates from rule-based planning metrics.

# Benchmarks / results
On Navhard, the paper reports 45.5 EPDMS with a V2-99 backbone; on HUGSIM it reports the best public-scenario zero-shot result among listed learning-based methods; standard Navtest performance is strong but not the overall best.

# Relation to the score-based E2E AD lineage
ZTRS stays firmly inside the Hydra/GTRS score-based lineage, but changes the training objective rather than the planner architecture.

# Caveats
- Still relies on a pre-enumerated trajectory action set.
- Uses offline rule-based rewards rather than fully interactive RL.
